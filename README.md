# 🛒 Inventory Management Chatbot

This is an AI-powered inventory chatbot built using n8n.

## 🚀 Features
- Manage inventory using chat
- Add / remove items
- Automation using workflows

## 📂 Files
- workflow.json → main chatbot logic

## ⚙️ How to Use
1. Import workflow.json into n8n
2. Activate workflow
3. Start chatting

## 🧠 Tech Used
- n8n
- AI automation
- JSON workflows
